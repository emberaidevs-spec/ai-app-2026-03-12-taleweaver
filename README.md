> Tale Weaver is an AI-powered story and poem generator that creates unique and engaging content based on user input. It can generate stories, poems, or even entire scripts, making it a valuable tool for writers, students, and anyone looking for creative inspiration.

## ✨ Features

- Generate stories based on prompts
- Create poems with specific themes or styles
- Allow users to customize tone, genre, and length

## 🛠 Tech Stack

- Frontend: Pure HTML / CSS / JavaScript
- Backend: Vercel Serverless Functions (Node.js)
- AI: Groq API (Llama 3.3 70B)
- Groq AI API via Vercel serverless function + JavaScript browser APIs for client-side rendering
- Dark theme with glassmorphism UI
- Fully responsive (mobile + desktop)
